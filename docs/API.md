# API Documentation

## エンドポイント一覧

### 男性ユーザーエンドポイント

- **GET /api/male-users/** - ユーザーのリストを取得
- **POST /api/male-users/** - 新しいユーザーを登録
- **GET /api/male-users/{id}/** - 特定のユーザーの情報を取得
- **PUT /api/male-users/{id}/** - 特定のユーザーの情報を更新
- **DELETE /api/male-users/{id}/** - 特定のユーザーを削除
