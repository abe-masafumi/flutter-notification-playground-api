# API Documentation

## エンドポイント一覧

### 男性ユーザーエンドポイント

- **GET /api/maleuser/** - ユーザーのリストを取得
- **POST /api/maleuser/** - 新しいユーザーを登録
- **GET /api/maleuser/{id}/** - 特定のユーザーの情報を取得
- **PUT /api/maleuser/{id}/** - 特定のユーザーの情報を更新
- **PATCH /api/maleuser/{id}/** - 特定のユーザーの情報を更新
- **DELETE /api/maleuser/{id}/** - 特定のユーザーを削除
